Melhorias Implementadas no Projeto

Funcionalidades Adicionadas

Indicador de Carregamento:

Adicionado um indicador visual (spinner) para informar o usuário enquanto os dados de personagens ou filmes estão sendo carregados.


Exibição Detalhada na Modal:

Detalhes dos personagens são exibidos em uma modal estilizada e responsiva.
Adicionada navegação fluida para exibir a modal apenas após os dados serem carregados.


Trailer dos Filmes:

Integração com URLs de trailers do YouTube para exibição dinâmica dos trailers correspondentes aos filmes.


Listagem Dinâmica de Personagens:

Criada uma lista interativa dos personagens de cada filme, que exibe os detalhes de cada personagem ao ser clicada.


Listagem De filmes:

Adicionado poster dos filmes para melhorar identificação visual.


Botão de tema para pagina:

Adicionado botão para alternar entre tema claro e escuro na pagina conforme preferência do usuário.
